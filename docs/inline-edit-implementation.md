## 🚀 使用指南

### 快速开始

1. **启动应用：** `npm run dev`
2. **进入详情页：** 点击任意代办卡片
3. **开始编辑：** 点击"编辑此待办"按钮
4. **编辑内容：** 在详情页内直接修改
5. **保存退出：** 点击"保存"按钮或等待自动保存

### 编辑模式功能

**可编辑字段：**
- 📝 **标题**：大尺寸输入框，支持快速编辑
- 🎯 **状态**：待办、进行中、已完成、暂停
- ⭐ **优先级**：低、中、高优先级
- 🏷️ **标签**：添加/删除标签，快速选择历史标签
- 📄 **内容**：富文本编辑器，支持格式化、图片等

**编辑工具栏：**
- 💾 **保存按钮**：立即保存并退出编辑模式
- ❌ **取消按钮**：取消编辑，恢复原始内容
- 📊 **状态指示**：显示自动保存状态

### 快捷键支持

**键盘快捷键：**
- `E` 键 - 快速切换编辑模式
- `Ctrl+S` - 手动保存（在编辑模式下）
- `ESC` - 取消编辑（在编辑模式下）
