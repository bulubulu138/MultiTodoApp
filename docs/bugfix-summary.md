## 🔧 修复 1：文件重命名功能

### 问题描述
**现象：** 修改代办标题时，Markdown文件名不同步更新，只修改了YAML frontmatter中的title字段

### 根本原因
`FileStorageManager.updateTodo()` 方法只更新文件内容，没有同步更新文件名

### 解决方案
- 添加标题变更检测逻辑
- 实现原子性文件重命名操作
- 同步更新UUID映射表
- 自动重命名相关附件文件

### 实现特性
✅ **智能检测：** 自动识别标题变更
✅ **原子操作：** 确保数据一致性
✅ **冲突解决：** 处理文件名冲突
✅ **附件处理：** 批量重命名相关文件
✅ **错误恢复：** 完整的回滚机制

---

## 🔧 修复 2：MiniSearch重复ID错误

### 问题描述
**现象：** 在卡片模式下编辑代办内容时报错：`MiniSearch: duplicate ID xxx`

### 根本原因
`FileIndexer.updateTodo()` 直接调用 `addTodo()`，导致MiniSearch尝试添加已存在的ID

### 解决方案
- 采用Remove + Add模式更新索引
- 添加静默删除方法避免异常
- 增强错误处理和日志记录

### 实现特性
✅ **幂等操作：** 支持重复更新
✅ **原子性：** 确保索引状态一致
✅ **静默处理：** 避免不必要的异常
✅ **详细日志：** 便于调试和监控
