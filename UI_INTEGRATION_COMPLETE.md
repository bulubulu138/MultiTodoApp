## 功能流程

### 用户操作流程
1. 用户点击工具栏的"同步"按钮
2. 打开SyncModal对话框
3. 自动启动WebSocket服务器（端口53318）和UDP广播（端口53317）
4. 显示"等待移动端连接..."状态

### 同步流程
1. **等待连接** - 桌面端广播设备信息，等待移动端发现
2. **配对验证** - 移动端连接后，桌面端生成6位配对码（大字号显示，5分钟倒计时）
3. **同步进行** - 配对成功后，显示进度条和文件传输状态
4. **同步完成** - 显示统计信息（发送、接收、跳过的文件数），2秒后自动关闭

---

## UI状态机

```
waiting (等待连接)
  ↓ (移动端连接)
pairing (显示配对码)
  ↓ (配对成功)
syncing (显示进度)
  ↓ (同步完成)
complete (显示统计) → 2秒后自动关闭
  
  ↓ (任何错误)
error (显示错误信息，提供重试按钮)
```
